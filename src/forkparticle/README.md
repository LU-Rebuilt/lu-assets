# ForkParticle Formats

Formats from the ForkParticle system used by LEGO Universe for particle effects.

| Directory | Format | Description |
|-----------|--------|-------------|
| [psb/](psb/) | PSB (.psb) | Binary emitter definitions — colors, velocities, sizes, textures |
| [effect/](effect/) | Effect (.txt) | Plain-text effect definitions referencing one or more PSB emitters |
