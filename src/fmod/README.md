# FMOD Audio Formats

Formats from the FMOD audio engine used by LEGO Universe.

| Directory | Format | Description |
|-----------|--------|-------------|
| [fev/](fev/) | FEV (.fev) | Event hierarchy — banks, categories, events, layers, envelopes, sound definitions |
| [fsb/](fsb/) | FSB (.fsb) | Sound banks — encrypted audio samples (MP3, ADPCM, PCM) |
