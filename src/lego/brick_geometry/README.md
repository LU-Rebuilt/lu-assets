# Brick Geometry

Parser for LEGO `.g` brick primitive geometry files. Each file contains vertex positions, normals, and triangle indices for one brick primitive at a specific LOD level.

Located in the client at `res/brickprimitives/lod0/`.

References: [lu-toolbox](https://github.com/Squareville/lu-toolbox) for Blender .g import, [lcdr/lu_formats](https://github.com/lcdr/lu_formats) g.ksy specification.
