# LEGO Formats

LEGO brick model formats used by LEGO Universe.

| Directory | Format | Description |
|-----------|--------|-------------|
| [lxfml/](lxfml/) | LXFML (.lxfml, .lxf) | XML brick models from LEGO Digital Designer |
| [brick_geometry/](brick_geometry/) | .g files | Binary brick primitive geometry (vertices, normals, indices) |
| [brick_assembly/](brick_assembly/) | — | Brick assembly pipeline: LXFML → renderable meshes with colors |
