# NetDevil Zone Formats

Formats that define game world zones and their content:

| Directory | Format | Description |
|-----------|--------|-------------|
| [luz/](luz/) | LUZ (.luz) | Zone layout — scenes, paths, transitions, spawn points |
| [lvl/](lvl/) | LVL (.lvl) | Scene data — chunked format with objects, environment, particles |
| [terrain/](terrain/) | Terrain (.raw) | Heightmap terrain with texture/color/blend maps |
| [lutriggers/](lutriggers/) | Triggers (.lutriggers) | XML trigger event/command definitions |
| [aud/](aud/) | AUD (.aud) | XML audio configuration per scene |
| [zal/](zal/) | ZAL (.zal) | Zone asset lists (plain text) |
| [ast/](ast/) | AST (.ast) | Asset lists with "A:" prefix (plain text) |
