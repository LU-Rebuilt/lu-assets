# NetDevil Archive Formats

Asset packaging and compression formats:

| Directory | Format | Description |
|-----------|--------|-------------|
| [pk/](pk/) | PK (.pk) | NDPK packed archives with table of contents |
| [sd0/](sd0/) | SD0 | Segmented zlib compression (used inside PK) |
| [pki/](pki/) | PKI (.pki) | Pack index — CRC-to-pack file mapping |
