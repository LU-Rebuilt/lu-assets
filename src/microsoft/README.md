# Microsoft / DirectX Formats

Standard Microsoft and DirectX formats used by LEGO Universe.

| Directory | Format | Description |
|-----------|--------|-------------|
| [dds/](dds/) | DDS (.dds) | DirectDraw Surface textures (DXT1/DXT3/DXT5 compressed) |
| [tga/](tga/) | TGA (.tga) | Targa raster images |
| [fxo/](fxo/) | FXO (.fxo) | Precompiled Direct3D 9 effect shaders |
| [stb/](stb/) | — | stb_image implementation (TGA/image loading backend) |
