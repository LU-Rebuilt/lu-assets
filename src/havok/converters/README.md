# HKX Converters

Conversion utilities between HKX and other formats:

| File | Description |
|------|-------------|
| `nif_to_hkx.h/cpp` | Convert NIF mesh geometry → HKX collision data (ExtendedMesh + Mopp wrapper) |
| `hkx_geometry.h/cpp` | Extract world-space triangle meshes from parsed HKX data (for rendering/navmesh) |
